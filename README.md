# antd-vue-admin

#### 介绍
vue3+ts 构建的后台管理系统。

#### 软件架构
1.  vue3:前端框架;
2.  ts:js的超集;
3.  antd-vue:UI框架;
4.  antv-g2:图表可视化;
5.  vite:前端新一代工具链;
6.  pinia:Vue 的存储库;
7.  axios:一个基于 promise 网络请求库

#### 安装教程
1.  git clone https://github.com/AJun-Kang/antd-vue-admin.git
2.  国内镜像：git clone https://gitee.com/kang-Junkai/antd-vue-admin.git

#### 使用说明

1.  初始化： npm install
2.  运行：   npm run dev
2.  打包：   npm run build
